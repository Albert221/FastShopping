
Быстрые Покупки - это мобильное приложение, разработанное с оглядкой на простоту. Оно позволяет вам составить простую коллекцию продуктовых списков, которыми вы можете легко управлять. Приложение не содержит рекламы. Предназначено для всех, кто хочет иметь простое приложение для составления списков покупок. Для использования приложения не требуются технические навыки.
